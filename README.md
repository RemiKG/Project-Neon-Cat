# The Neon Cat

A reskinned physics puzzle game built for short demo moments: clean white maze lines, glow-heavy visuals, and quick tool-based interactions.

## What it does
- You control a glowing cat-ball through maze-style levels.
- You get 6 physics-style tools: Heat, Cold, Mass, High Pressure, Vacuum, and Tunneling.
- Tools can push, pull, warp motion, or temporarily phase through walls.
- Includes normal + hacker visual modes, particles, trails, and audio.

## Run it
- `npm run start` and open `http://localhost:3000`

## Quick checks
- `npm run check`
- `npm run verify:levels`
- `npm run verify:assets`
